1) Run scriptOnce.html file once.
2) Signup page.html is for the user to sign in.
3) ProductDetails.html is where the admin can view,edit,delete or add new products.
4) productsDISPLAY.html is the page where the product details along with image are display. There is a feature to add an item to the cart and view the cart.
5) cart.html is the cart page where quantity can be modified and the subtotal can be viewed.
